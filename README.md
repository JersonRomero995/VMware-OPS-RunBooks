This repository contains my personal runbooks, automation scripts, and design notes
for VMware technologies, including:

- vSphere (ESXi, vCenter, storage, networking)
- NSX-T

The purpose of this repo is to:

1. Document real-world troubleshooting procedures.
2. Build reusable automation scripts.
3. Serve as a long-term knowledge base.
4. Demonstrate expertise in VMware + Cloud + Automation.
